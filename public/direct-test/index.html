Shered Testing Tools

Welcome to the testing tools for Shered. These tools help you test various features of the application.

Route Deviation Testing

Tools for testing the route deviation notification feature. All test trips have notifications enabled by default. When a significant deviation is detected, the system sends email notifications to group members.

  • Create Test Trip
    Create a new test trip with future dates and "planning" status for testing route deviation.
  • Test Route Deviation
    Test the route deviation by sending simulated GPS coordinates. If coordinates are >5km off the route and notifications are enabled, an email will be sent.
  • Legacy Deviation Test
    The original test page for route deviation.
